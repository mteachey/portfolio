Aggregate Materials

Description

Aggregate Materials is a custom Wordpress site developed for a large, international construction materials company.

My Role

  • Front-end Developer
  • Website Manager
  • UX Advisor

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • API
  • AJAX
  • jQuery

Process

This is a custom Wordpress site with custom plugins, calls to several APIs and lots of user interactivity to make a content heavy site easy and enjoybale to use.

I was the lead developer for building this twenty page, interactive site.

Some of the challenges to overcome with this site were due to the budget for the site and the large number of locations (over 700) and separate operating companies (thirteen) that made up the corporation. Many of the cost saving plugins we used for different functionality - such as the location map, timeline, and operating company page - required lots of customization and clever hacking.

The company also wanted dynamic stock information, current market cap and stock price, as well as, news postings. This required making an API call to retrive and display most current values for this data.

The designer and I created a beautiful UI with all the functionality the client wanted while also launching the site on time and on budget.

Aggregate Materials is an alias and there is no live link to this site to respect a NDA.

Future Versions

The "Next Page" buttons were added to the site after user-feedback testing. The "Careers" page and the and some of the substainability content was added during stage 2 of this site, as well as, a few tweaks to the main menu.
