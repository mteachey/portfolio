Process

Weekend Roulette is app that allows users to leave the weekend decision making to the app. The app provides two day activities and two evening ones with the 'spin of a wheel' (click of a button).

The app makes calls to four APIs - a hiking trail API, biking trail API, latitude/longitude API, and a restaurant API - in order to obtain the activity selections for users.

Users experience some simple CSS animations that add to the enjoyment and experience of the app.

Technologies used : jQuery, JavaScript, CSS, HTML, AJAX

Future Versions

Future versions of the app will add more APIs, such as “points of interest” from Google Maps and music from Bands in Town, to add more types of activities. In addition, future versions will include an improved restaurant API and using Google Geocaching for latitude and longitude.
