Process

Mountz Jewelers wanted a more modern, polished site for their eCommerce website. The refresh was developed on the Shopify platform.

This was my first time customizing a Shopify theme which is built using Liquid and Sass.

Some of the challenges with developing this site was creating the home page testimonial area. The theme did not come with this type of section. In order to make it work with the component structure of a Shopify theme while still having the functionality the client wanted, I refactored the blog component to use as a testimonial one. There were also some custom pricing conditionals that had to be created in the catalog page template files to work with their unique pricing structure.

Future Versions

The initial launch of the site refresh focused on the home page, catalog pages, and overall theme. The next stage will be reorganizing the interior pages, writing new copy, and giving them a more polished, professional look.
