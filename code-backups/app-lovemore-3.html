Love More Laugh More

Description

Love More Laugh More is a full-stack, single-page application to motivate and inspire users to care for themselves more. The app provides a place for users to set goals, see and celebrate their progress, and find ideas for inspiration.

Process

Love More Laugh More is a full-stack React app. I began by deciding on the schema for the database, then I determined the components for and created the static version of the client-side React app. The REST API was developed using Node.js and Express and includes endpoints for creating and reading (CRUD) information in the database. The database was set up using PostgreSQL. Mocha and Jest were used for testing.

For the client-side app, I wanted to include visual representations of the data and I wanted them to be dynamic, so that when new data was added they would automatically update. I used the Nivo library to display a bar graph of user progress towards their goals, a pie graph displaying of the data of the different types of activities, and a line graph displaying user's entered energy level and moods for the month.

I also used styled components to add an animated svg.

Future Versions

Future versions of the app will include a form for users to enter their target goals for the week, such as yoga 3 times, and then display those on the dashboard for users to check off rather than filling out the form every day with activities. Then, the daily form will be for recording daily gratitudes and the user's mood, as well as, a place to enter aha moments (a feature idea from user feedback).
