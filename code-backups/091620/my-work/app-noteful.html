Noteful

Description

Noteful is minimally styled full-stack, single-page application to save and organize notes. The client-side app was built with React and the server-side app was built using Node.js and PostgreSQL.

Process

This application allows users to create new notes and folders. The user can view all the notes, notes by folder, or individual notes. Users can also delete notes.

The front end of this application was developed using React. I created a static version of the app first deciding on the best components and component hierarchy for the different features required for the application. Once a static version was created, I added state and functionality to the app.

The back-end was created using Node.js, Express and PostgreSQL. The REST API was organized with Express routes and service objects. There are endpoints for all CRUD operations - creating new notes and folder, reading notes and folders (all and by id), updating notes and folders, and deleting notes. Tests were set up for all of the routes and different possible errors using Mocha and Jest.

Future Versions

Future versions of the app will allow users to delete folders and update exisiting notes and folders.
